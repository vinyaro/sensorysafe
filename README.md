# Execute e implante seu aplicativo do AI Studio

Este contém tudo que você precisa para executar o aplicativo localmente.

## Executar Localmente

**Pré-requisitos:**  Node.js


1. Instale as dependencias: `npm install`
2. Defina a `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com sua chave de API do Gemini
3. Execute o aplicativo: `npm run dev`
